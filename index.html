Quick Installation

Get started in minutes with our simple setup process

1

Prerequisites

Ensure you have the required software installed

macOS 14.0 or later
Xcode 15.0 or later
Ollama (optional)
2

Clone & Build

Download and build the project

git clone https://github.com/your-repo/fraud-dms
cd fraud-dms
open FMLD\ Panel.xcodeproj
3

Configure

Set up your environment variables

export APP_ENV=production
export OPENAI_API_KEY=your_key
export STRIPE_PUBLISHABLE_KEY=your_key
4

Run

Launch the application and start monitoring

# Build and run in Xcode
# Or use command line:
./FraudDMS-Panel
